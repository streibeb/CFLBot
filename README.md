## Setup

A config.json file must be created in the src/ directory with the following format:

```json
{
    "cfl": {
        "baseUri": "http://api.cfl.ca",
        "apiKey": ""
    },
    "reddit": {
        "clientId": "",
        "clientSecret": "",
        "userAgent": "",
        "refreshToken": ""
    },
    "logging": {
        
    },
    "interval": 1
}
```
